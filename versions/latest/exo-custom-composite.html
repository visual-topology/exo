Exo - Custom Elements - Composite Controls

Custom HTML elements described here wrap some useful structures for building UIs with exo

Merge Lists


<exo-merge-lists id="test_merge" label="Test"
     value='[["aaa","ccc"]]' tooltip="Help!!!"
     size="5" options1='[["aaa","AAA"],["bbb","BBB"]]'
     options2='[["ccc","CCC"],["ddd","DDD"]]'/>
<script>
    var btn = document.getElementById("test_merge");
    btn.addEventListener("exo-value",e => { console.log(e.detail); })
</script>
