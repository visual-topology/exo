Exo

Exo is a CSS mini-framework for building simple but stylish web applications. It is based on the typography of and incorporates code from Skeleton CSS and, like Skeleton, is licensed under the open source MIT license.

Exo adds colours, icons from the material design icon set, custom styled input controls, and more...

This document provides documentation based on examples for Exo.

Installing Exo

To use exo, add the following html5 code to your document's <head>: 

        <link href='//fonts.googleapis.com/css?family=Raleway:400,300,600' rel='stylesheet' type='text/css'/>
        < href="https://cdnjs.cloudflare.com/ajax/libs/skeleton/2.0.4/skeleton.min.css" rel="stylesheet" type="text/css"/>
        <link rel="stylesheet" href="exo.css"/>
        <!-- icon files are optional -->
        <link rel="stylesheet" href="icons/exo-icons.css"/>
