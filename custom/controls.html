Exo - Custom Elements - Controls

Custom HTML elements described here wrap some useful structures for building UIs with exo

Use standard event listeners 

        var btn = document.getElementById("test_button");
        btn.addEventListener("click", ()=>btn.setAttribute("label","!!!!"));
